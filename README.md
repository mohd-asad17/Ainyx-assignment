# ReactFlow Canvas – App Graph Builder

A responsive App Graph Builder built using React, TypeScript, ReactFlow (XYFlow), Zustand, TanStack Query, and shadcn/ui.

## Features

- Interactive ReactFlow canvas
- Drag, select, connect, and delete nodes
- Dotted canvas background
- Responsive layout
- App selector with graph switching
- Service Node Inspector
  - Status badge
  - Config/Runtime tabs
  - Editable node name
  - Synced slider and numeric input
- Mock API integration using TanStack Query
- Zustand for UI and application state management

## Tech Stack

- React + Vite
- TypeScript
- ReactFlow (@xyflow/react)
- TanStack Query
- Zustand
- shadcn/ui
- Tailwind CSS

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
